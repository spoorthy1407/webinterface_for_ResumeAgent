# PDF Upload & Export - Quick Start Guide

## 30-Second Overview

Your AI Resume Agent now supports:
- ✅ PDF resume uploads
- ✅ Smart content extraction & analysis
- ✅ Download in TXT, HTML, or Word format
- ✅ No information mixing or loss

## Getting Started

### Upload Your Resume (30 seconds)

**Option 1: Drag & Drop**
1. Find "Drop your resume file here" area
2. Drag PDF onto it
3. Wait for extraction (2-5 seconds)

**Option 2: Click to Browse**
1. Click the upload area
2. Select PDF or TXT file
3. Click "Open"

### What Happens Next

1. System extracts text from PDF
2. Analyzes resume structure
3. Loads data into editor
4. Shows formatted preview

### Download Your Resume

1. Click "Download" button
2. Hover to see options:
   - **TXT File**: Plain text for job applications
   - **HTML File**: Styled format for email/web
   - **Word (.docx)**: Edit in Microsoft Word
3. Click desired format
4. File downloads instantly

## File Format Guide

### TXT - Best for Job Applications
- ATS-friendly format
- Works with all systems
- Smallest file size
- Download: `YourName_Resume.txt`

### HTML - Best for Email/Web
- Professional appearance
- Clickable links
- View in any browser
- Download: `YourName_Resume.html`

### DOCX - Best for Editing
- Open in Microsoft Word
- Edit and customize
- Share with colleagues
- Download: `YourName_Resume.docx`

## Troubleshooting

### PDF Won't Upload
**Problem**: File not accepted or extraction fails
**Solution**: 
- Ensure PDF has selectable text (not scanned image)
- Try another PDF reader first
- Try TXT format instead

### Can't See Downloaded File
**Problem**: File not appearing in Downloads folder
**Solution**:
- Check browser download folder
- Check Downloads directory in File Explorer
- Try different format
- Check browser download settings

### Resume Looks Wrong
**Problem**: Sections missing or mixed up
**Solution**:
- Original PDF might have formatting issues
- Try TXT export to see plain content
- Review original PDF in another reader
- Contact support with sample file

### Export Button Not Working
**Problem**: Download doesn't start
**Solution**:
- Check pop-up blocker settings
- Try different browser
- Try incognito/private window
- Check available disk space

## Common Questions

**Q: Is my resume data safe?**
A: Yes! All processing happens on your device. Nothing is uploaded to servers.

**Q: What formats are supported?**
A: Upload PDF or TXT. Export as TXT, HTML, or DOCX.

**Q: Can I edit the downloaded file?**
A: Yes! DOCX format is fully editable in Word.

**Q: Which format should I use?**
A: Use TXT for applications, HTML for email, DOCX for editing.

**Q: How long does extraction take?**
A: Usually 2-5 seconds for typical resume PDFs.

**Q: Can I upload multiple files?**
A: One at a time currently. Upload, download, then upload next.

**Q: Will my formatting be preserved?**
A: Content and structure yes, visual formatting depends on target format.

**Q: What if PDF has scanned images?**
A: System will notify you. Use OCR tools first, then upload.
