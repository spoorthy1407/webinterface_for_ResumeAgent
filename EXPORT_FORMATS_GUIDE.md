# Resume Export Formats Guide

## Overview

The AI Resume Agent provides three professional export formats to suit different needs:

## Format Comparison

| Aspect | TXT | HTML | DOCX |
|--------|-----|------|------|
| **Best For** | ATS Systems | Web/Email | Microsoft Word |
| **File Size** | Small (~2-5 KB) | Medium (~5-10 KB) | Medium (~10-20 KB) |
| **Formatting** | Plain text | Styled CSS | Basic formatting |
| **Editability** | Text editor | Browser/Email | MS Word |
| **Compatibility** | 100% universal | 100% universal | Windows/Mac/Online |
| **Print Quality** | Basic | Professional | Professional |
| **Supports Links** | No | Yes | Yes |

## TXT Format (Plain Text)

### Characteristics
- Pure text format with no special encoding
- Optimal for Applicant Tracking Systems (ATS)
- Maximum compatibility across all systems
- Smallest file size
- Professional structure with clear sections

### Structure
```
JOHN SMITH
====================

john@example.com | (555) 123-4567 | New York, NY
LinkedIn: linkedin.com/in/johnsmith | GitHub: github.com/johnsmith

PROFESSIONAL SUMMARY
-----------
Experienced software engineer with 5+ years...

SKILLS
-----------
Technical: Python, JavaScript, React, Node.js, PostgreSQL
Soft Skills: Leadership, Communication, Problem-solving
Domain Knowledge: Web Development, Cloud Architecture

PROFESSIONAL EXPERIENCE
-----------
Senior Developer at Tech Corp
2022-Present | San Francisco, CA
Led development of microservices architecture...
  • Improved performance by 40%
    (Reduced latency from 500ms to 300ms)
  
...
```

### Best Use Cases
- Submitting to job boards
- Uploading to ATS systems
- Email applications
- Systems that don't support rich formatting

### How to Download
1. Click "Download" button in preview
2. Select "TXT File"
3. File saves as `YourName_Resume.txt`

## HTML Format (Web/Email)

### Characteristics
- Professional styling with embedded CSS
- Maintains visual hierarchy and formatting
- Responsive design for different screen sizes
- Perfect for email signatures or web portfolios
- Clickable links for contact information

### Features
- Clean, modern design
- Easy to view in web browsers
- Compatible with Gmail, Outlook, etc.
- Print-friendly styling
- Dark mode compatible

### Visual Elements
- Centered header with name
- Contact information with icons
- Color-coded section headers
- Proper typography and spacing
- Print optimization

### Best Use Cases
- Email to hiring managers
- Web-based application portals
- Personal website embedding
- Portfolio or blog display
- Print to PDF for sharing

### How to Download
1. Click "Download" button in preview
2. Select "HTML File"
3. File saves as `YourName_Resume.html`
4. Open with any web browser

## DOCX Format (Word Document)

### Characteristics
- Microsoft Word compatible format
- Basic OpenXML structure
- Editable in Word, Google Docs, Office Online
- Professional formatting preserved
- ATS-friendly despite rich format

### Compatibility
- Microsoft Word (2010+)
- Google Docs
- Microsoft 365 Online
- LibreOffice
- Apple Pages

### Editing Features
- Full text editing support
- Simple formatting changes
- Font and size adjustments
- Add comments and track changes
- Export to other formats from Word

### Best Use Cases
- Submit to companies requesting DOCX
- Further editing in Word
- Share with colleagues for feedback
- Collaboration in Office 365
- Convert to PDF from Word

### How to Download
1. Click "Download" button in preview
2. Select "Word (.docx)"
3. File saves as `YourName_Resume.docx`
4. Open with Microsoft Word or compatible software

## Which Format to Use?

### Use TXT If:
- Submitting through ATS system
- Company doesn't specify format
- Simple, clean presentation needed
- Maximum compatibility required
- Smaller file size needed

### Use HTML If:
- Emailing directly to hiring manager
- Posting on web platforms
- Need professional styling
- Want clickable links
- Sharing via web link

### Use DOCX If:
- Company requests Word format
- Need to edit before submitting
- Want advanced formatting options
- Collaborating with others
- Using in Office environment

## Export Tips

### Before Exporting
1. **Review Content**: Check all information is accurate
2. **Verify Formatting**: Ensure no missing sections
3. **Check Spelling**: Proofread before export
4. **Update Dates**: Confirm all dates are current

### After Exporting
1. **Verify File**: Open downloaded file to check
2. **Test ATS**: Run TXT version through ATS checker
3. **Share Preview**: Send HTML to colleagues
4. **Keep Backup**: Save original in multiple formats

## Technical Details

### File Naming
All exports use consistent naming:
```
[First]_[Last]_Resume.[extension]

Examples:
- john_smith_resume.txt
- jane_doe_resume.html
- alex_johnson_resume.docx
```

### Metadata
- TXT: Plain text, no metadata
- HTML: Includes character encoding UTF-8
- DOCX: Basic document properties

### Encoding
- TXT: UTF-8 plain text
- HTML: UTF-8 with HTML5 declaration
- DOCX: UTF-8 with XML declaration

## Troubleshooting

### File Won't Open
**TXT Issue**: 
- Any text editor should open
- Try Notepad, VS Code, or online viewer
- Check file isn't corrupted

**HTML Issue**:
- Try different browser (Chrome, Firefox, Safari)
- Check file isn't corrupted
- Verify CSS loaded properly

**DOCX Issue**:
- Update Microsoft Office
- Try Google Docs or Office Online
- Use LibreOffice if Word unavailable

### Formatting Issues
**TXT**: No formatting supported
**HTML**: Styling applied automatically
**DOCX**: Basic formatting only, some loss possible

### Download Problems
- Check browser allows downloads
- Verify disk space available
- Disable pop-up blocker
- Try different browser
- Try incognito/private window

## Advanced Usage

### Batch Downloads
Want all formats at once?
1. Export as TXT
2. Export as HTML
3. Export as DOCX
4. Keep all in resume folder

### Format Conversion
- HTML → PDF: Print/Export from browser
- DOCX → PDF: Save As in Word
- TXT → HTML: Use converter tool
- DOCX → TXT: Save As option in Word

### Custom Edits
1. Export as DOCX
2. Open in Word
3. Make edits
4. Save modified version
5. Export to other formats if needed
